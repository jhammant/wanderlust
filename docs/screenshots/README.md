# Wanderlust Screenshots

This directory contains realistic mock demo screenshots showing what the Wanderlust UI looks like.

## Demo HTML

[View the interactive demo](demo.html) to see:

- **Travel DNA** - Your travel profile with statistics
- **Interactive Map** - All your trips visualized with pins
- **Travel Timeline** - Year-based trip history
- **AI Recommendations** - Personalized destination suggestions
- **Road Trip Planner** - Complete itinerary planning tool

## Features Demonstrated

1. **Trip Map**: Interactive Leaflet map with pins for each destination
2. **Timeline View**: Year-by-year trip history
3. **AI Recommendations**: 5 curated destination suggestions with reasoning
4. **Road Trip Planner**: Multi-day road trip planning tool
5. **Travel DNA Statistics**: Your travel patterns and preferences

## Technologies Used

- HTML5, CSS3
- Vanilla JavaScript
- Leaflet.js for interactive maps
- Responsive design
